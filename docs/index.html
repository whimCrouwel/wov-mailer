Features

必要な機能がぜんぶ入ってる

シンプルだけど、実際のメール配信業務で使えるように設計されています。

🗂

Airtable 連携

ベース・テーブルを選んでフィルターをかけるだけ。既存の連絡先管理をそのまま活用できます。

✏️

差し込み変数

エディタで @ を入力するとフィールド候補が表示。{{name}} などをそのまま本文に挿入できます。

👁

送信前プレビュー

Airtable の実データで差し込み結果を確認。受信者リストも名前・メールアドレスで一覧表示できます。

🎨

HTML テンプレート

Gmail・Outlook・Apple Mail など主要クライアントで表示確認済みのテンプレートを5種類内蔵。

📋

送信履歴

過去の配信履歴を管理。当時の送信先スナップショットを保存するので、Airtable が変わっても確認できます。

🔁

Clone & 再送

過去のブロードキャストをワンクリックで複製。件名や本文を調整してすぐ再送できます。

● Claude Code MCP Integration

Claude がアプリを直接操作する

wov-mailer には MCP（Model Context Protocol）サーバーが内蔵されています。Claude Code をターミナルで使うと、自然言語の指示だけで Compose 画面のフィルター・テンプレート・件名・本文をすべて設定できます。送信は必ず人間が確認してから。

自然言語で設定できる

フィルターやテンプレートの選択を日本語で指示するだけ。Claude が 10 種類の MCP ツールを使って Compose 画面をリアルタイムに更新します。

🔒

送信は必ず人間が確認

MCP に「送信する」ツールは存在しません。Claude はメールを準備するだけで、実際の送信ボタンを押すのは常にあなた自身です。

⚡️

アプリを開いたまま会話できる

内蔵ターミナルから直接 Claude Code を操作。ブラウザに切り替える必要なし。設定変更が画面に即座に反映されます。

初回のみ一度だけ登録が必要です：

claude mcp add wov-mailer \
  --transport http http://127.0.0.1:3741 \
  --scope user
Use Case

Tally フォーム → Airtable → メール配信

問い合わせや登録フォームから、自動でリストに追加して配信まで。配信停止リンク付きで、丁寧なメール体験を作れます。

📝

Tally フォーム

問い合わせ・登録フォームを作成。回答が自動で Airtable に蓄積されます。

tally.so
🗂

Airtable

連絡先を一元管理。unsubscribed チェックボックスで配信停止フラグを管理できます。

airtable.com
✉️

wov-mailer

unsubscribed = false でフィルターして対象者だけに送信。配信停止リンクをメール本文に含めます。

wov-mailer
🚀

Resend

高い到達率でメール配信。送信結果は履歴に保存され、いつでも確認できます。

resend.com

📝 フォーム → Airtable の連携

Tally の「Airtable integration」を使えば、フォームの回答が送信されるたびに Airtable のレコードが自動追加されます。Make / Zapier 経由でも連携可能。

🚫 配信停止の仕組み

メール本文に Tally フォームの URL（例：配信停止フォーム）を含めます。回答があったら Airtable の unsubscribed を true に更新。次回配信時はフィルターで除外されます。

⚡ Claude で一括準備

内蔵ターミナルで「先週登録した人にウェルカムメールを送って」と一言伝えるだけ。Claude が Airtable の条件を設定して Compose 画面を準備します。あとは送信するだけ。

🔁 定期配信のフロー例

月次ニュースレターなら：① Tally で新規登録受付 → ② Airtable でリスト管理 → ③ wov-mailer でフィルター送信 → ④ 配信停止はフラグで管理。すべてノーコードで完結。

How to use

かんたん4ステップで送信完了

難しい設定は不要。初回だけ API キーを登録すれば、あとは選んで送るだけです。

1

API キーを設定

Settings 画面で Airtable の Personal Access Token と Resend の API キーを入力。設定はローカルに保存され、リポジトリには含まれません。

2

送信先を絞り込む

Compose 画面でベース・テーブルを選択し、フィルター条件を追加。チェックボックスや選択肢フィールドにも対応しています。

3

本文を書いてプレビュー

@ でフィールドを差し込みながら本文を作成。プレビューで実際の表示を確認し、「View list」で受信者を最終確認。

4

送信

「送信する」ボタンを押すだけ。送信結果は履歴に自動保存されます。
